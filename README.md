# ⚽ Predicción de Marcadores - Mundial FIFA 2026

Aplicación web para predecir resultados de partidos del Mundial 2026 usando dos modelos:

- **Bayesiano Jerárquico** (Baio-Blangiardo)
- **XGBoost** con features de Elo y forma reciente

## Características
- Selector de cualquier selección nacional
- Dos modelos independientes lado a lado
- Heatmaps de probabilidades
- Top 10 marcadores exactos
- Estadísticas de los equipos (Elo, forma reciente)
